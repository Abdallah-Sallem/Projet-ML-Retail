# Analyse Comportementale Clientèle – Atelier Machine Learning

## Description du projet

Ce projet est réalisé dans le cadre de l’atelier pratique du module Machine Learning (GI2 – 2025/2026).

L’objectif est d’analyser le comportement des clients d’un site e-commerce de cadeaux à partir
d’une base de données réelle et imparfaite, afin de :

- Comprendre les profils clients
- Segmenter la clientèle
- Prédire le churn (départ des clients)
- Extraire des informations utiles à la prise de décision marketing

Le projet couvre l’ensemble de la chaîne de traitement en data science :
**Exploration → Préparation → Modélisation → Évaluation → Déploiement**.
